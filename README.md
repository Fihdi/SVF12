# 8hp 12dB/Oct Voltage Controlled state-variable Filter

![](https://github.com/Fihdi/SVF12/blob/main/SVF12Front.png?raw=true)

A simple VCF with 2 Audio and CV Inputs, Ping input, Variable Resonance and Oscillator mode with V/Oct tracking. The filter core naturally produces Lowpass, Bandpass and Highpass versions of the input signal. By subtracting the Bandpass signal from the input signal a Bandstop response is achieved.

The module can be switched from Filter mode (Switch in the FILT position) to Oscillator mode (Switch in the OSC position). In Oscillator mode the module will produce clean sine waves on the BP, BS and LP outputs.

A trimmer on the back of the module can be found to tune the Volt/Octave tracking.
